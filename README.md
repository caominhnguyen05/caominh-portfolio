# Cao Minh Nguyen — Portfolio Website

This project features my portfolio website built with Next.js, Tailwind CSS, Framer Motion, and responsive design.

You can access the website at: [Portoflio Link](https://caominhnguyen.vercel.app/)

## 🚀 Tech Stack

- **Frontend:** Next.js, React, TypeScript, Tailwind CSS, Framer Motion
- **Tools:** Git, GitHub, Vercel, VS Code
- **Design:** Responsive UI, motion transitions

## 💼 Portfolio Description

1. **Homepage** with dynamic sections: intro, experience, projects, contact.
2. **Project Showcase** with live links and GitHub repositories.
3. **Modern UI** using dark theme, gradients, responsive layout, and smooth scroll animations.

## 🛠️ How to Run Locally

Clone the repository:

```bash
git clone https://github.com/caominhnguyen05/caominh-portfolio.git
cd caominh-portfolio
```

Install dependencies and run locally:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## 📂 Repository Structure

- `src/app/page.tsx` — main landing page composition
- `src/app/components/` — reusable intro, experience, projects, contact sections
- `src/app/layout.tsx` — global metadata and font/theme setup
- `public/` — static assets and icons
